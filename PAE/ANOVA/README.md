# A análise de variância (ANOVA)

Esta análise pode determinar se as médias de dois ou mais grupos são diferentes. A ANOVA usa testes F para testar estatisticamente a igualdade entre médias. Neste post, mostrarei como a ANOVA e os testes F funcionam, usando um exemplo de ANOVA com um fator.

Mas espere um minuto... você já parou para pensar por que você usaria uma análise de variância para determinar se as médias são diferentes? Também mostrarei como as variâncias fornecem informações sobre as médias.
